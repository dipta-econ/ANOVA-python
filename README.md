# ANOVA-python
One-way ANOVA implementation using Python

## Objective
To test whether there is a significant difference among three treatment means.

## Method
Implemented one-way ANOVA using scipy.stats.f_oneway().

## Hypothesis
H0: μ1 = μ2 = μ3  
H1: At least one mean differs

## Result
The p-value was less than 0.05, so the null hypothesis was rejected.

## Tools Used
- Python
- Pandas
- SciPy
